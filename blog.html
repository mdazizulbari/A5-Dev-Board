Question-1: What are the different ways to select an element in the DOM?

We can use document.getElement properties to select an element in the DOM. There about 5 of them, but the commonly used ones are getElementbyId, getElementsbyClassName, getElementsbyName etc.

Question-3: What is event delegation in the DOM?

Making code more efficient using event bubbles is called event delegation.

Question-4: What is event bubbling in the DOM?

I understood it but it's hard to explain. If we assign a event to any HTML tag, it will trigger in nested way, it will start from document, than html, than body and so on. When it gets to the particular tag, it will again come back in the same reverse nested way. That way is called event bubbling.
